# 手書きノート内の参考文献略記

このファイルは，手書きノートに現れる略記，色付き矢印，欄外メモを参考文献候補へ対応させるための台帳である。
プロジェクトごとに必ず更新する。

## 基本方針

- 略記を見つけても，推測だけで本文に引用しない。
- まずこの対応表に候補を登録し，`references/REFERENCE_INDEX.md` と照合する。
- ページ番号・定理番号は，実際に確認できた場合だけ書く。
- 色付き矢印の先に数学的主張が書かれている場合は，参考文献候補ではなく本文補足候補としても記録する。
