# 参考文献

このディレクトリには，補助参考文献 PDF と読解メモを置く。

- `pdfs/`: 論文，講義ノート，書籍抜粋などの PDF。
- `notes/`: 各参考文献の要点，使った定理番号，本文との対応メモ。
- `REFERENCE_INDEX.md`: 現在置かれている PDF 群の棚卸しと要確認事項。

原稿本文に参考文献由来の内容を入れる場合は，文献名，節番号，定理番号，ページ番号，URL などを `notes/` または `REFERENCE_INDEX.md` に記録してから本文へ統合する。

## 運用原則

- 参考文献は本文を補強するために使う。主テキストとユーザー作成ノートの構成を置き換えない。
- 引用箇所，ページ番号，定理番号，出版情報を推測で書かない。
- 本文中の注釈では，実際に使った箇所のページ番号・定理番号・節番号を具体的に書く。
- PDF 末尾の参考文献一覧では，ページ番号・定理番号ではなく書誌情報を中心にまとめる。
- 新しい参考文献を追加したら，まず `REFERENCE_INDEX.md` に登録し，必要に応じて `notes/` に読解メモを作る。
- Git 管理できない PDF は，台帳上で「ローカル保管」と明記する。
- 100MB を超える共有 PDF は，Git LFS を前提にせず，意味のある単位で分割して各ファイルを 100MB 未満にする運用を先に検討する。

詳細な探し方は `notes/README.md`，末尾参考文献欄の書き方は `notes/final_bibliography_style.md`，手書きノート由来の略記は `notes/handwritten_reference_aliases.md` に従う。
