# 修正文候補

このファイルは，独立査読やユーザー確認が必要な修正文候補を一時的に記録する場所である。
通常査読で根拠が確定しており，方針や辞書と衝突しない修正は，本文へ直接反映してよい。

## 運用

- 独立査読で出た候補は，本文へ直接反映せず，まずここに記録する。
- ユーザーが採用した候補は本文へ反映し，このファイルから削除する。
- ユーザーが不要と判断した候補も削除する。
- 候補文には，対象箇所，理由，テキストからどれだけ離れているか，優先度を書く。

- 優先度: 高 / 中 / 低
- 根拠: [使用テキスト，ノート，参考文献，独立査読]
- テキストからの距離: 小 / 中 / 大
