# 用語辞書

このファイルは，用語・表現の統一に使う。
非標準表現やユーザー固有の好みを使う場合は，ここに記録する。

## 承認済み表現

| 概念 | 採用する表現 | 避ける表現・注意 |
| --- | --- | --- |
| 一意性 | 一意的な | 一意な，は原則避ける |
| 同型の一意性 | 同型を除いて一意的に定まる | 一意的同型を除いて一意的に定まる |
| 図式 | 次の図式を可換にする，次の可換図式を得る | 三角図式が可換，四角図式，重複図式 |
| 整合性 | 整合的にする | 整合する，は文脈に注意 |
| 自然性 | 自然な写像，自然な射影，自然な同型 | 自然同型は別概念として扱う |
| 商への写像 | 商写像 | 射影と混同しない |
